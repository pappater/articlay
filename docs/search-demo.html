üéÆ How to Use

  1. Open the main Articlay interface
  2. Look for the search bar below the header
  3. Type your search query (source name or keyword)
  4. Use the dropdown to filter by specific fields if needed
  5. Results update automatically as you type
  6. Click "Clear" to return to all articles
